Real-Time Posture & Fatigue Monitor System

"This project detects user posture in real time using computer vision and tracks fatigue levels through continuous monitoring. The system alerts users when poor posture or high fatigue is detected to improve health and productivity."

Problem Statement

Prolonged bad posture while using computers causes health issues like back pain, fatigue, and reduced productivity. Existing solutions lack real-time monitoring and automatic feedback. This project provides a smart posture and fatigue monitoring system using webcam-based computer vision.

Literature Review

  • Traditional posture monitoring uses wearable sensors like accelerometers and gyroscopes.
  • These systems provide accurate tracking but are expensive and uncomfortable.
  • Computer vision approaches using OpenCV and MediaPipe enable non-contact posture monitoring.
  • Previous research focuses mainly on pose estimation and gesture recognition.
  • Existing fatigue detection systems rely on eye-tracking or physiological sensors.
  • Most solutions lack integrated posture and fatigue monitoring.
  • Real-time alert systems are limited in current research.

Research Gap / Innovation

  • No integrated posture and fatigue monitoring solution.
  • Limited real-time intervention systems.
  • Many solutions require specialized hardware.
  • Lack of cost-effective health monitoring tools.
  • Need for non-invasive ergonomic monitoring systems.

Proposed Solution

  • Real-time posture detection.
  • Fatigue scoring.
  • Automated health alerts.
  • Webcam-based monitoring without additional hardware.

System Methodology

Dataset / Input

Live webcam video captured using OpenCV and processed using MediaPipe body landmark detection.

Model / Architecture

Computer vision pipeline using MediaPipe for posture detection, fatigue tracking algorithm, and real-time alert system.
